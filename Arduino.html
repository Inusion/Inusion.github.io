For this project, we had many issues, but overall, the outcome was exactly what we desired. We made a Simon Says color game, in which it procedurly generates a random order of colors for you to tap certain buttons to. If you do it incorrectly, it will end the game, and if you succeed it proceeds to the next round. One of the issues we had (and still have) is the Blue LED, we couldn't get any blue LED that we wired up to tun on, and couldn't figure out why. But it still plays the noise for it, so it works just fine. The other issue was getting the order to work, because when you pressed it, it counted as a click for how long it was pressed, so we needed a thing that disabled the button while it was being pressed, but not when it was first pressed. Our project also had active memory, so it stored the highscores you got.
